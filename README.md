# Algorithms Visualizer
This is a simple application that visualizes some of the most common sorting algorithms. The application is built using C#.
The following algorithms are implemented:
1. Bubble Sort
1. Selection Sort
1. Insertion Sort
1. Merge Sort
1. Quick Sort
1. Heap Sort
1. Radix Sort
1. Counting Sort
1. Bucket Sort
